depoya yazma (senkron)

Defter şifreli bir dosya olarak GitHub deposunda durur; depo herkese açık olsa da içerik parolasız okunamaz. Yazabilmek için bir GitHub anahtarı gerekir. Anahtar yalnız bu tarayıcıda, şifrelenmiş olarak saklanır.

anahtar nasıl alınır?
  1. github.com → sağ üstte profil → Settings
  2. En altta Developer settings → Personal access tokens → Fine-grained tokens → Generate new token
  3. Ad: kara kapli, süre: 1 yıl (en uzunu seç)
  4. Repository access: Only select repositories → veranda-araclari
  5. Permissions → Repository permissions → Contents: Read and write
  6. Generate token de, github_pat_... ile başlayan metni buraya yapıştır
